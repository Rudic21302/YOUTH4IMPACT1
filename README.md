# YOUTH4IMPACT 🌍

Frontend for a youth impact platform — built with SCSS for a clean, scalable, and maintainable stylesheet architecture. The project demonstrates modern CSS practices and structured frontend development for a purpose-driven web presence.

---

## Overview

YOUTH4IMPACT is a platform designed to connect and empower young people around social impact initiatives. This repository contains the frontend implementation with a focus on clean SCSS organization and responsive design.

---

## Tech stack

- **HTML5** — semantic, accessible markup
- **SCSS** — modular stylesheet architecture with variables, mixins, and nesting
- **CSS3** — animations, flexbox/grid layout

---

## SCSS structure

The project uses a component-based SCSS approach:
- Variables for consistent color and typography tokens
- Mixins for reusable layout patterns
- Nested rules that mirror the HTML component hierarchy

This keeps styles scalable and easy to maintain as the project grows.

---

## How to run

For development with SCSS compilation:

```bash
# Clone the repository
git clone https://github.com/Rudic21302/YOUTH4IMPACT1.git
cd YOUTH4IMPACT1

# Compile SCSS (requires sass CLI or Node.js sass)
sass --watch styles.scss:styles.css

# Open index.html in a browser
```

Or simply open `index.html` directly if CSS is already compiled.
